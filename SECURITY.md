# Security Policy

## Supported Versions

Only the latest published version of `radar-devtools` and the desktop app receives security updates.

| Version | Supported          |
| ------- | ------------------ |
| Latest  | :white_check_mark: |
| Older   | :x:                |

## Reporting a Vulnerability

If you discover a security vulnerability, **please do not open a public GitHub issue**.

Instead, report it privately by emailing:

**lucasfigueiredo285@gmail.com**

Please include:

- A description of the vulnerability and its impact
- Steps to reproduce (proof of concept if possible)
- The affected version(s)
- Any suggested fix or mitigation

You can expect:

- Acknowledgement within 72 hours
- An initial assessment within 7 days
- A coordinated disclosure timeline once the issue is confirmed

## Scope

In scope:

- The `radar-devtools` SDK on npm
- The Radar desktop application (Electron)
- The MCP server exposed by the desktop app

Out of scope:

- Vulnerabilities in third-party dependencies (please report those upstream)
- Issues that require physical access to the developer's machine
- Self-inflicted misconfiguration of MCP clients

Thank you for helping keep Radar and its users safe.
